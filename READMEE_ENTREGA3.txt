## Instrucciones para abrir el proyecto
- Descargar Xampp
- Ubicar el proyecto en la carpeta htdocs que se encuentra dentro de la carpeta Xampp
- abrir Xampp e iniciar los servidores apache y MySQL
- Descargar el archivo que contiene la base de datos (127_0_0_1.sql)
- en Xampp abrir admin de MySQL
- en la base de datos buscar el apartado "importar"
- Subir el archivo 127_0_0_1.sql

Realizados los pasos anteriores la pagina deberia funcionar correctamente, para ingresar en el buscador colocar http://localhost/login-registro/
